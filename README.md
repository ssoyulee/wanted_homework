## 실행 방법
```
  # git 소스를 받아온다.
  $ git clone https://github.com/ssoyulee/wanted_homework.git
  
  # docker 폴더로 이동한다.
  $ cd wanted_homework/docker
  
  # docker-compose를 실행한다.
  $ docker-compose up -d 
```

## 접속 방법
### DB Data init
```
http://{host}/init
```

### Swagger URL
